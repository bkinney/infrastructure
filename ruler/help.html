Overview

The Architect's Scale Tutorial has been in use since 2007 as a standalone tool for introducing students to the use of a triangular architectural scale (ruler). The tutorial contains 3 sections: Reading Feet, Reading Inches, Demonstration, and Practice. Each section is interactive, and each one provides opportunities for students to test their understanding and receive immediate feedback. With this LTI release, grade passback has been added so that instructors can know which of their students have been successful in completing the tutorial.

An assignment based on this tutorial is always pass/fail in the sense that students MUST successfully measure 3 consecutive lines before they can submit a score. All submitted scores are 100%. Students who can not complete the tutorial due to technical concerns will need an alternative assignment, such as a paper and pencil exercise. This tutorial is not recommended in educational settings where students are unlikely to have access to a computer with Flash capabilities and at least a 17" monitor.

Step-by-Step

  • Import from Canvas Commons (recommended)
    1. Log into Canvas
    2. In the global menu (far left), locate and click the Commons button.
    3. Now you are at the Canvas Commons. Search for the Architect's Scale module. Click the title text.
    4. Click the green "Import into Course" button. This will create a copy of the entire module, including the tutorial assignment.
    5. Select the course into which you wish to import the module.
    6. Modify the module and assignment in any way you like.
      1. Set assignment points, category, due date, release date, etc.
      2. Set module requirements, etc.
      3. Be sure both assignment and module are published before notifying students about this content.
  • Deploy as an External Tool Assignment
    1. NOTE: An assignment containing the tutorial is automatically included by the module import from Canvas Commons. Use these steps only if you wish to create a second assignment, or if you have chosen to install the tutorial without the module (below).
    2. Create a Canvas Assignment with a submission type as 'External Tool'
    3. Optionally check the "Load this Tool in a New Tab" checkbox (recommended).
    4. Set other assignment options as desired and publish the assignment.
    5. When students visit the assignment page, they will see the tutorial. To recieve a grade, they must complete the Practice section by accurately measuring 3 lines in a row using the virtual scale. Completing other sections is optional, and will not impact their grade.
  • Install standalone tutorial
    1. Note: installation is NOT required or recommended if you import the module.
    2. Obtain a key and secret.
    3. Launch Canvas, and browse to any course for which you have instructor access.
    4. Click on the Apps tab.
    5. Locate and click the Architect's Scale button. If the button is missing, or you are not in Canvas, search the full EduAppCenter. Follow the instructions from the appropriate tutorial.
    6. Click the blue Add Tool button.
    7. In the window that pops up, enter the key and secret.

FAQ

Click on a question to reveal its answer

  • What LMS's can be used with this tutorial?
    • The tutorial should work in any LMS that supports the IMS LTI v1.1 standards, however, the only way to know for sure is to try. The University of Delaware tests only against Canvas. Please note that the tutorial itself was created with Flash, and should not be expected to work on mobile devices.
