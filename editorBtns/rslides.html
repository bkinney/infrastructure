UD Hosted Applications | Documentation (technical) | EduAppCenter

Academic Technology Services hosts a number of in-house Learning Tool Interactivity (LTI) applications. Although some of these tools will function within the context of any LTI consumer, and a few are available as standalone applications, the majority rely on Canvas-specific LTI extensions and/or the Canvas Application Programming Interface (API) and are only useful within Canvas. Sakai@UD users are encouraged to consider switching to Canvas if any of the tools described on this site are of interest.

For more information about the UD hosted applications, follow the UD Hosted Applications link above. For a list of other LTI applications, or to learn more about Learning Tool Interoperabilty, visit the EduAppCenter.
