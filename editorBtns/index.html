UD Canvas Editor Buttons

 
Name Description Availability Type
jQuery Widgets Add dynamic UI widgets to a Canvas page (tabs, accordion, dialog, glossary, self-test) Available only in jQuery subaccount pre-installed in IT>ATS sub-account Editor Button
iFrameIT add an iframe containing Canvas File or 3rd party (secure) content to a Canvas page by request Editor Button
UDCapture embed Embed a UDCapture lecture into a Canvas page pre-installed Editor Button

These are all anonymous apps and should not require much maintenance. The only dependent file is /canvas/oembed/index.php, which is required for the jQuery Widgets. The custom css and js files for the UD account must also be in place.

 

Currently there is no online storage location for the most recent versions...I have them on my chromebit. When I decide where they should live, I will put the link here.
